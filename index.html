About

TimeTrackerFD is a single-page application designed to track work time in small teams.

It is built using vanilla JavaScript, HTML and CSS, with a modular JavaScript architecture and no external frameworks.

The application works on both desktop and mobile devices and is developed as part of the PP5 university project.

TimeTrackerFD is a free alternative to paid time tracking tools, focused on simplicity, transparency and ease of use.

The project was designed and implemented by Daniel Jakubas and Filip Cetnarowicz, and is deployed on Microsoft Azure.
